# PDF分割ツール

PDFの各ページを左右または上下に分割するWebアプリです。  
pdf-libによるクライアントサイド処理のため、ファイルサイズ制限なし・サーバーへのアップロードなしで動作します。

## 機能

- ドラッグ＆ドロップ または クリックでPDFを選択
- **分割方向**: 自動（横長→左右 / 縦長→上下）、左右、上下から選択
- **分割位置**: スライダーで1〜99%の任意の位置を指定（デフォルト50%）、リセットボタンで50%に戻せる。左端が「左｜上」側、右端が「右｜下」側
- **ページ範囲**: 指定したページのみ分割、それ以外はそのまま出力（例: `1-3,5,7-9`）
- 処理はブラウザ内で完結（PDFはサーバーに送信されない）

## デプロイ

### Vercel

```bash
vercel
```

`index.html` 1ファイルのみのため、zero-configでそのまま配信されます。

### GitHub + Vercel（推奨）

1. このリポジトリをGitHubにpush
2. [vercel.com](https://vercel.com) でリポジトリをインポート
3. 設定不要でデプロイ完了 → 以後pushするたびに自動デプロイ

## ローカル確認

```bash
npx serve .
# → http://localhost:3000
```

または `index.html` をブラウザで直接開いても動作します。

## 技術スタック

| 用途 | ライブラリ | 配信 |
|------|-----------|------|
| PDF操作 | [pdf-lib](https://pdf-lib.js.org/) v1.17.1 | CDN (unpkg) |
| スタイル | [Tailwind CSS](https://tailwindcss.com/) | CDN |

ビルドツール・バックエンド・依存関係のインストールは不要です。

## 元スクリプトとの対応

このWebアプリは `split_pdf.py`（Tkinter製デスクトップツール）を元に作成しました。

| デスクトップ版 (Python/pypdf) | Web版 (JavaScript/pdf-lib) |
|---|---|
| ページ向き自動判定で分割 | 同じロジックを移植 |
| 分割位置（%）指定 | スライダーで指定 |
| ページ範囲指定 | テキスト入力で指定 |
| ファイルダイアログ | ドラッグ＆ドロップ |
| 分割後に自動オープン | ダウンロードボタン |
| — | 分割方向の手動選択（追加機能） |
